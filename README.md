# Applied Cryptography in Rust

A collection of cryptographic implementations, protocol experiments, and mathematical notes written in Rust.

This repository focuses on bridging rigorous mathematical thinking with practical cryptographic engineering.

## Topics

- finite field arithmetic
- elliptic curve cryptography
- digital signatures
- key exchange protocols
- zero-knowledge systems
- cryptographic protocol design
- post-quantum cryptography

## Repository (possible) Structure 

```text
classical/          Classical cryptographic constructions
elliptic-curves/   Finite fields and ECC
protocols/         Cryptographic protocols
zk/                Zero-knowledge experiments
notes/             Short mathematical and implementation notes
```

## Goals

- study modern cryptography through implementation
- develop reliable and testable cryptographic code
- explore the mathematical foundations of secure systems
- experiment with protocol engineering in Rust

## Status

This repository is an evolving long-term project focused on applied cryptography and secure system design.
