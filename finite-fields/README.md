# Finite Fields

A small Rust library implementing arithmetic over prime finite fields.

This crate is part of the broader *Applied Cryptography in Rust* project and serves as a foundational layer for later cryptographic constructions such as elliptic curve cryptography, digital signatures, and zero-knowledge proof systems.

## Goals

- implement finite field arithmetic from first principles
- write clear and testable mathematical code
- explore the algebraic foundations of modern cryptography

## Planned Features

- modular addition and subtraction
- modular multiplication
- modular inversion
- exponentiation
- finite field abstractions
- property-based tests
